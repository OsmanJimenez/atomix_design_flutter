# Atomix Design System Example

This example demonstrates how to use the Atomix Design System in a Flutter application.

## Features Demonstrated

- Theme configuration (light/dark modes)
- Button variants and sizes
- Text fields with icons
- Cards with different variants
- Chips with selection states
- List tiles
- Badges with different variants
- Spacing and layout using design tokens

## Running the Example

```bash
flutter pub get
flutter run
```
